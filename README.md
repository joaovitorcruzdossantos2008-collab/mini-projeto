# mini-projeto

Projeto: Gerenciamento de Produtos em Java com ArrayList e Herança
Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO), utilizando ArrayList, herança, construtores e sobrescrita de métodos.
Funcionalidades:
Classe Produto com atributos nome, preco e quantidade
Construtor com parâmetros para inicialização dos atributos
Método toString() sobrescrito para exibição formatada dos dados
Uso de ArrayList<Produto> para gerenciamento de objetos
Adição de produtos na lista
Recuperação de produtos por índice
Impressão do tamanho da lista
Classe ProdutoPerecivel herdando de Produto
Uso de super() para inicialização de atributos herdados
Atributo adicional dataValidade
Impressão de objetos e listas no console
Tecnologias:
Java • POO • ArrayList • Herança • Construtores • Polimorfismo
